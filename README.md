## prerequisites

- nasm
- rust and target `i389-unknown-none-elf`
- qemu

## build and run

1) clone the repository:
```
git clone https://github.com/uktgh/EucalyptusOS.git
cd EucalyptusOS
```

2) build the project:
```
make
```

3) run the os in qemu:
```
make run
```
